# talocode-xsearchlane (Python)

Python SDK package placeholder for XSearchLane.

## Planned

```bash
pip install talocode-xsearchlane
```

```python
from xsearchlane import XSearchLaneClient

client = XSearchLaneClient()  # TALOCODE_API_KEY
result = client.search(query="AI agent sandbox deleted files")
```

Hosted routes:

- `POST /v1/xsearchlane/search`
- `POST /v1/xsearchlane/research`
- `GET /v1/xsearchlane/health`
